# FormattingOptions
        
::: details 原文链接
https://microsoft.github.io/monaco-editor/docs.html#interfaces/languages.FormattingOptions.html
:::

用于格式化模型的接口

```ts
interface FormattingOptions {
    insertSpaces: boolean;
    tabSize: number;
}
```

## insertSpaces
- 类型：`boolean`
- 描述：喜欢空格而不是制表符。
## tabSize
- 类型：`number`
- 描述：选项卡的大小（以空格为单位）。
