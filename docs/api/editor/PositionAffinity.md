## Left
- 值：`0`
- 描述：最左边的位置。

## LeftOfInjectedText
- 值：`3`
- 描述：如果给定的位置在注入的文本上，则首选其左侧的位置。

## None
- 值：`2`
- 描述：没有偏好。

## Right
- 值：`1`
- 描述：最右边的位置。

## RightOfInjectedText
- 值：`4`
- 描述：如果给定的位置在注入的文本上，则首选其右侧的位置。
