参数提示的配置选项

```ts
interface IEditorParameterHintOptions {
    cycle?: boolean;
    enabled?: boolean;
}
```

## cycle
- 类型: `boolean`
- 描述：启用参数提示的循环。默认为false。
## enabled
- 类型: `boolean`
- 描述：启用参数提示。默认为true。
