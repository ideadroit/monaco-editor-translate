概览标尺位置的描述。



```ts
interface OverviewRulerPosition {
    height: number;
    right: number;
    top: number;
    width: number;
}
```

## height
- 类型: `number`
- 描述：概览标尺的高度
## right
- 类型: `number`
- 描述：概览标尺的右边距
## top
- 类型: `number`
- 描述：概览标尺的顶部边距
## width
- 类型: `number`
- 描述：概览标尺的宽度
