# OverlayWidgetPositionPreference
        
::: details 原文链接
https://microsoft.github.io/monaco-editor/docs.html#enums/editor.OverlayWidgetPositionPreference.html
:::

## BOTTOM_RIGHT_CORNER
- 值：`1`
- 描述：将覆盖小部件放置在右下角

## TOP_CENTER
- 值：`2`
- 描述：将覆盖小部件定位在顶部中央

## TOP_RIGHT_CORNER
- 值：`0`
- 描述：将覆盖小部件放置在右上角
